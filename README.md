# ⛽ Gas Station AI Assistant

An AI-powered customer service chatbot built for a retail gas station. 
Customers can ask questions about services, products, fuel, and more — 
and get instant accurate answers 24/7.

## 🚀 Features
- Answers customer questions instantly using AI
- Trained on real gas station data
- Clean chat interface built with Streamlit
- Powered by LangChain and Groq LLM
- Uses RAG (Retrieval Augmented Generation) for accurate answers

## 🛠️ Tech Stack
- Python
- LangChain
- Groq LLM (llama-3.3-70b)
- HuggingFace Embeddings
- Chroma Vector Database
- Streamlit

## ⚙️ How to Run

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Create a .env file and add your Groq API key:
   GROQ_API_KEY=your-key-here
4. Run the app:
   streamlit run app.py

## 💡 Use Case
Small businesses spend hours answering the same customer questions repeatedly.
This chatbot automates that process — saving time and improving customer experience.
